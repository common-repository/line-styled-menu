== Description ==

You can easily add these amazing effects in your Wordpress installation by adding our plugin. With "Line Styled Menu" you can add your desired effect with a shortcode. The shortcode must contain two mandatory parameters. The first is the menu which will be used for the effect and the second is the effect itself.

The demo and article is on http://tympanus.net/codrops/2016/01/06/inspiration-for-line-menu-styles/

The following effects are available so far:
- sebastian
- prospero
- viola
- antonio
- miranda
- ariel

More:
http://www.cookforweb.com



== Installation ==

1. Download the plugin and add it in your plugin's folder.
2. Activate the plugin



== Instructions ==

Add the shortcode [line_menu menu="SELECT MENU" type="SELECT EFFECT"] in your header file of your theme
